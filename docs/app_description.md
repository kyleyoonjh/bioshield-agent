# AI신약 (AiRemedy) 앱 설명

No virus, No Cancer — AI신약은 바이러스와 암으로부터 사람의 건강을 지키겠다는 목표로 태어난 AI 기반 신약 개발 어시스턴트입니다. AlphaFold·ESMFold 구조예측, AutoDock Vina 도킹, ADMET 분석, mRNA 암백신 신항원 설계까지 모든 결과는 가상의 수치가 아닌 실제 계산과 PubMed·ClinicalTrials.gov·UniProt·ChEMBL·PubChem 등 공개 데이터베이스에 기반합니다. 연구자가 전문 용어 없이 자연어로 질문하면 AI 에이전트가 스스로 도구를 선택해 표적 발굴부터 후보물질 스크리닝, 임상 근거 정리까지 전 과정을 안내합니다. 궁극적 목표는 신약 개발의 문턱을 낮춰 더 많은 바이러스성 질환과 암을 정복하고, 인류의 건강한 삶에 실질적으로 기여하는 것입니다.

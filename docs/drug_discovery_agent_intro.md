# 🧪 대화형 AI 신약개발 어시스턴트 (Drug Discovery Assistant)

## 한 줄 소개
연구 목표를 한 문장으로 설명하면, AI가 실제 단백질 구조를 확보하고 승인 약물 라이브러리를 실제로 도킹·스크리닝한 뒤, 결과가 기준에 미달하면 스스로 원인을 분석해 전략을 바꿔 재시도하고, 완료 후에는 문헌·임상·타겟·화합물·구조개선·종합평가까지 실제 데이터로 이어서 답해주는 상시 대기형 연구 파트너입니다.

---

## 핵심 기능

**1. 자연어 한 마디로 설계 시작 — 실제 구조·도킹 파이프라인 자율 실행**
"SARS-CoV-2 스파이크 단백질을 억제할 수 있는 승인된 약물을 찾아줘" 같은 문장 하나면 충분합니다. 에이전트는 AlphaFold DB 조회(또는 ESMFold 실시간 예측)로 실제 타겟 구조를 확보하고, 그 구조의 실제 좌표로 도킹 박스(리셉터)를 준비한 뒤, 큐레이션된 승인 약물 라이브러리를 AutoDock Vina로 실제 도킹·스크리닝합니다. 특정 리간드를 지정하면 단일 도킹 모드로, 목표만 말하면 스크리닝 모드로 자동 전환됩니다.

**2. 결과 미달 시 자가 성찰(Reflection) 후 자동 재시도**
Vina를 사용할 수 없거나 생존 후보가 기준에 못 미치면, Planner → Evaluator → Reflector 루프가 원인을 구조화해 분석하고 전략(고속/표준/정밀)을 전환해 최대 3회까지 재시도합니다. 각 회차의 판정과 성찰 내용, 전략 전환 이유가 Decision Log에 그대로 기록됩니다.

**3. 실시간 한국어 진행 상황 중계 (6단계, 병렬 처리로 가속)**
타겟 구조 확보 → 리셉터 준비 → 도킹/스크리닝 → 필터링 → 랭킹 → AI 해설까지 6단계가 대화창에 한국어로 실시간 표시됩니다. 스크리닝은 후보 2개를 동시에 실제 병렬 도킹하도록 개선되어(측정값: 후보당 평균 62초 → 35초, 약 1.8배), 22개 후보 전체 스크리닝 시간이 실측 기준 크게 단축되었습니다.

**4. 완료 후 결과 카드 + AI 해설 + 추천 다음 행동**
스크리닝이 끝나면 순위·결합친화도·drug-likeness·ADMET이 포함된 후보 카드가 즉시 나타나고, AI가 그 결과를 해설하면서 이 결과에서 실제로 유의미한 다음 행동(결합 포켓 분석, 구조 개선, 종합 평가 리포트 등)을 데이터에 근거해 추천합니다 — 예를 들어 후보에 PAINS 구조 경보가 있으면 그 사실을 근거로 결합 포켓 분석을 우선 추천합니다.

**5. 6개의 실제 후속 연구 에이전트 — 대화로 바로 호출**
| 에이전트 | 실제 데이터 소스 |
|---|---|
| 📚 관련 논문 | PubMed 실시간 검색 |
| 🏥 임상시험 현황 | ClinicalTrials.gov 실시간 검색 |
| 🎯 타겟 질병 연관성/경로 | UniProt DISEASE 코멘트 + Reactome 경로 |
| 🧫 화합물 검색 | PubChem 유사 화합물 / ChEMBL 실측 IC50 억제제 |
| 🧬 결합 포켓 분석 | 실제 도킹 포즈 좌표 + AlphaFold PAE 신뢰도 |
| 🧪 구조 개선 (SAR) | 실제 생물학적 등가체 치환 + 재도킹 비교(예측이 아닌 실측) |
| 📊 종합 평가 리포트 | 투명한 공식의 우선순위 점수 + 개발 위험도 + 근거 인용 |

한국어로 물어봐도 됩니다 — "결핵 논문 찾아줘"처럼 한국어 질환/병원체명은 실시간으로 영어 검색어로 변환된 뒤 조회됩니다.

**6. VCF/FASTA 업로드 (체세포 변이 / 단백질 서열)**
VCF를 업로드하면 Ensembl VEP로 변이를 실시간 재검증(파일 자체의 GENE 라벨을 그대로 믿지 않음)해 실제 유전자를 확정하고 동일한 스크리닝 파이프라인으로 이어갑니다. FASTA 서열을 올리면 ESMFold로 구조를 예측해 스크리닝합니다.

**7. 한 타겟에 집중하는 대화 — "🆕 새 연구" 버튼으로만 초기화**
한 세션에서 타겟이 한 번 정해지면, 사용자가 "🆕 새 연구" 버튼을 누르기 전까지 다른 타겟으로 임의 전환되지 않습니다. 완료된 작업에 대한 후속 질문은 항상 그 작업의 실제 데이터에 근거해 답합니다.

## 실제로 나오는 결과
- 순위가 매겨진 약물 후보 목록 (결합친화도 70% + drug-likeness 30% 가중 종합 점수)
- 실제 AutoDock Vina 결합친화도(kcal/mol) — Vina 미사용 시 "휴리스틱"으로 명확히 구분 표기
- 실측 ADMET 서브셋: Veber's Rule, PAINS 구조 경보(RDKit 공식 480종 카탈로그), 합성 접근성(SA score), 간독성 구조 경보
- 결합 포켓 잔기 목록, 수소결합/소수성 접촉 후보 (실제 도킹 포즈 좌표 기반), AlphaFold 구조는 PAE 신뢰도까지
- 실제 재도킹된 SAR 유사체와 결합친화도 변화(Δ)
- 다운로드 가능한 HTML/PDF 분석 리포트

모든 수치는 RDKit·AlphaFold DB·ESMFold·AutoDock Vina·PubMed·ClinicalTrials.gov·PubChem·ChEMBL·UniProt·Reactome의 실제 계산/조회 값이며, AI는 의도 파악·전략 선택·성찰·해설·추천에만 관여하고 새로운 과학적 수치를 지어내지 않습니다.

---

## 연구원에게 어떤 도움이 되는가

- **진입 장벽 제거** — UniProt ID나 도킹 파라미터를 몰라도 자연어 목표 한 문장이면 충분합니다.
- **실패해도 사람이 다시 설정할 필요 없음** — 기준 미달 시 에이전트가 원인을 분석해 전략을 자동 전환·재시도합니다.
- **결과에서 끝나지 않고 다음 단계로 이어짐** — 스크리닝 결과 하나로 끝나지 않고, 그 결과를 근거로 문헌·임상·구조개선·종합평가까지 실제 데이터로 계속 대화할 수 있습니다.
- **대기 시간 단축** — 병렬 도킹으로 실측 약 1.8배 빨라졌고, 6단계 진행 상황이 한국어로 실시간 표시됩니다.
- **신뢰할 수 있는 답변** — 모든 점수는 결정론적 엔진 계산값 또는 실시간 조회값에 근거하며, 무엇이 실제 도킹이고 무엇이 근사치인지 항상 명시적으로 구분됩니다.
